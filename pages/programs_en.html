• JSEQ_scRNAseq – a tool for analyzing single-cell sequencing data, enabling efficient processing and interpretation of results.
  • GEDSpy – an advanced Python library for quick retrieval and analysis of gene function data from various sources.
  • BioSRA – a program facilitating the download of large biological datasets from next-generation sequencing (NGS) from NCBI databases.
  • JIMG – a Python library for processing confocal microscope images, with annotation capabilities for further analysis using AI algorithms.
  • JBioSeqTools – a versatile Python library for preparing and processing genetic sequences for the creation of therapeutic viral plasmids.
  • SQL-db – a sample database project in Oracle SQL, designed for clinical research.
